这是一个非常关键的问题！**Animation Montage (蒙太奇)** 是连接“美术表现”和“程序逻辑”的桥梁。一个 Sequence 只是单纯的“播放录像”，而 Montage 则是“可交互的演出”。

以下是制作高质量 Montage 的标准流程和评判标准：

### 第一步：如何制作 Montage (基础操作)

1. **找到资源**：在内容浏览器中找到你的绿色图标的 **Animation Sequence** (例如 `Primary_Melee_A_Slow`)。
2. **创建**：右键点击它 -> **Create** -> **Create AnimMontage**。
3. **命名**：通常以 `AM_` 开头或 `_Montage` 结尾，例如 `AM_Attack_Combo1`。
4. **打开**：双击打开它，你会看到类似时间轴的编辑器。

------

### 第二步：评判 Montage 好坏的 4 个硬指标

一个“好”的战斗 Montage，必须配置好以下 4 点，否则它只是个花架子：

#### 1. Slot (插槽) —— **最容易出错的地方！**

- **要求**：Montage 里的 Slot Name 必须和你的动画蓝图 (AnimBP) 里的 Slot Node 一致。

- 检查方法

    ：

    - 在 Montage 编辑器中间，你会看到一行写着 `DefaultGroup.DefaultSlot`。
    - 如果你的 AnimBP 里用的是 `DefaultSlot`，那就没问题。
    - **常见坑**：Paragon 的资源有时候会默认用 `UpperBody`，如果你不改回 `DefaultSlot`，你的角色就会只动上半身，或者根本不动。

- **操作**：点击那个 Slot 名字，选择 `DefaultSlot`。

#### 2. Anim Notifies (动画通知) —— **灵魂所在**

这是 C++ 代码知道“什么时候该干什么”的唯一途径。你需要右键点击 `Notifies` 轨道来添加通知。

- **关键通知一：`ANS_Trace` (或类似的攻击判定)**
    - **位置**：在金箍棒挥出去、看起来最有力量的那一瞬间（比如第 15 帧到第 20 帧）。
    - **作用**：告诉 C++ “现在开启武器碰撞检测”。如果没有这个，你的棒子穿过敌人也不会造成伤害。
    - *注：如果你暂时没有自定义 Notify，可以先不管，我们后面可以用简单的 Timer 或通用的 Notify 来做。*
- **关键通知二：`SaveAttack` (连击窗口)**
    - **位置**：在动作快结束，但还没完全收招的时候。
    - **作用**：告诉系统“如果玩家现在按鼠标左键，就把它存下来，等当前动作播完立刻放下一招”。
    - **好坏标准**：如果窗口太早，玩家会觉得“我还没按你就打出去了”；如果太晚，玩家会觉得“手感很肉，按了没反应”。

#### 3. Root Motion (根运动) —— **打击感的来源**

- **现象**：有些攻击动作，角色会向前跨一步。
- **问题**：如果不用 Root Motion，角色模型向前跨了，但胶囊体（碰撞体）还在原地。打完一套动作，角色会“滑”回原点。
- **要求**：勾选左侧 Details 面板里的 **Enable Root Motion**。
- **好坏标准**：勾选后，你会发现角色真的“走”出去了，胶囊体也跟着动了。这会让战斗看起来非常扎实，没有“滑冰”的感觉。

#### 4. Blend In / Blend Out (融合时间)

- **位置**：Details 面板里的 `Blend Option`。

- 要求

    ：

    - **Blend In (进入时间)**：攻击通常要快，设为 `0.1` 或 `0.05`，让动作瞬间爆发。
    - **Blend Out (退出时间)**：如果设得太长，角色收招会很慢；设得太短，动作会突然跳回站立。通常 `0.2` 左右比较合适。

------

### 第三步：实战演练（以悟空的第一段攻击为例）

请尝试按照这个标准配置你的第一个攻击：

1. 打开 `AM_Attack_Combo1` (你刚创建的)。

2. **检查 Slot**：确保是 `DefaultSlot`。

3. **开启 Root Motion**：在左侧勾选 `Enable Root Motion`。

4. 观察动作

    ：

    - 拖动时间轴，找到棍子挥舞最猛烈的那一段。
    - 找到动作开始收招、准备恢复站立的那一刻。

5. **保存**。

### 接下来...

当你把 `Attack1`, `Attack2`, `Attack3` 的 Montage 都这样简单设置好（至少改好 Slot 和 Root Motion），然后**在蓝图里把它们填进去**。

你就可以进游戏按左键了！你会发现：

1. 角色真的打出来了。
2. 因为开启了 Root Motion，角色攻击时会自然地向前移动。

等你做完这一步，如果发现“打不到人”或者“连击不顺畅”，我们再来专门加 **Notifies**。先让它动起来！
